# Курсовая работа 1 курса Серова Леонида Александровича
## Программирование игровых приложений: Игровое приложение «Пятнашки».

### Настоящая курсовая работа выполняется посвящена разработке игрового приложения «Пятнашки» на языке C++. Тема работы: «Разработка классической игры “Пятнашки” с графическим интерфейсом и логикой решения на языке C++».
Объектом исследования является процесс разработки игрового приложения с использованием объектно-ориентированного программирования и библиотек для создания графического интерфейса.
Предметом исследования — методы реализации игровой логики, генерации решаемых конфигураций, визуализации игрового поля и взаимодействия с пользователем.
Цель работы — разработать полностью функциональное приложение «Пятнашки», включающее игровую механику, графический интерфейс, систему подсчёта ходов и возможность генерации решаемых игровых состояний.
### Для достижения цели поставлены следующие задачи:
•	Изучить основы разработки приложений на C++ с использованием графических библиотек (например, SFML, Qt или консольного интерфейса).
•	Реализовать игровую логику: перемещение костяшек, проверка завершения игры, подсчёт ходов.
•	Разработать алгоритм генерации решаемых игровых состояний.
•	Создать графический интерфейс для отображения игрового поля и взаимодействия с пользователем.
•	Протестировать приложение на корректность работы и удобство использования.

### Чем обусловлена актуальность работы?
Актуальность темы обусловлена сохранением интереса к классическим логическим играм, которые развивают пространственное мышление, логику и внимание. Игра «Пятнашки» является не только развлекательным, но и образовательным инструментом, используемым в педагогике для развития когнитивных способностей.
В современных условиях разработка подобных приложений позволяет студентам углубить знания в области объектно-ориентированного программирования, работы с графикой, алгоритмизации и тестирования. Кроме того, создание игровых приложений на C++ является востребованным направлением в индустрии разработки ПО, что повышает практическую ценность работы.
### Потенциальные пользователи
Разрабатываемое приложение ориентировано на следующие категории пользователей:
•	Студенты и школьники: для развития логического мышления и изучения основ программирования через игровую механику.
•	Преподаватели: как демонстрационный материал при изучении C++ и алгоритмов.
•	Любители классических игр: для развлечения и тренировки внимания.
### Описание работы программы
Приложение будет состоять из следующих модулей:
#### 1.	Игровой движок:
o	Управление состоянием игрового поля (матрицы 3×3, 4×4, 5×5).
o	Логика перемещения костяшек.
o	Проверка завершения игры.
o	Подсчёт ходов и времени.
2.	Генератор игровых состояний:
o	Алгоритм генерации решаемых конфигураций.
o	Возможность выбора уровня сложности.
#### 3.	Графический интерфейс:
o	Отображение игрового поля.
o	Кнопки управления (новая игра, сброс, выход).
o	Отображение счётчика ходов и времени.
#### 4.	Модуль сохранения и загрузки (опционально):
o	Сохранение текущей игры.
o	Загрузка предыдущих сессий.
#### 5.	Блоки меню:
o	Начать новую игру
•	Выбрать уровень сложности
o	Продолжить прошлую игру
o	Просмотреть историю прошлых игр
o	Посмотреть справочную информацию (инструкция и история игры)
o	Информация о создателе
o	Выйти из игры
Программа будет реализована с использованием объектно-ориентированного подхода, что обеспечит модульность, читаемость кода и возможность дальнейшего расширения.
